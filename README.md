Arduino Sequential Seven-Segment Display Control

This project controls a single-digit seven-segment display using Arduino.
The segments turn ON sequentially from OFF state, then turn OFF in reverse order.
The sequence repeats continuously with a fixed delay between each segment.

Objective:
To learn seven-segment interfacing, Arduino digital output control,
loop programming, and timing delays in embedded systems.

Applications:
LED pattern generation, display systems, embedded learning, animation effects.

Conclusion:
This project demonstrates sequential ON and OFF control of seven-segment display
using Arduino and helps understand basic embedded programming concepts.
